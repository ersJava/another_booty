Party Like It's 1999

Bulma | jQuery | AJAX

Test your 90's music knowledge with this interactive web browser game that dynamically displays the HTML with the jQuery library and pulls data from multiple APIs.

A timer will begin and lyrics of a song that existed in 1999 will begin to display on the screen. The user will try and guess the title of the song and submit their answer in the box below. Guessing correctly will unlock the information page with the music video, full song and lyrics, and photos of the Artist/Band.

Crystal Collector

HTML5 | CSS | jQuery

Crystal Collector

HTML5 | CSS | jQuery

An interactive web browsers game that dynamically updates the HTML with the jQuery library.

The player will be given a random number at the start of the game. There are four crystals displayed as buttons on the page and the value of each crystal is hidden until clicked. The player wins the game by matching the total score to random number. The game restarts whenever the player wins or loses and the value of each crystal will change.
